HTML（HyperText Markup Language，超文本标记语言）

一、HTML 简介

HTML（HyperText Markup Language，超文本标记语言）是用于创建网页的标准标记语言。它通过一系列标签（tags） 描述网页的结构和内容（如文本、图片、链接、表格等），浏览器会解析这些标签并将其渲染为可视化的网页。

  • 核心特点：
    • 不是编程语言，而是标记语言（通过标签定义内容的含义和结构）。
    • 标签通常成对出现（如 ``），由开始标签、内容和结束标签组成。
    • 不区分大小写，但推荐使用小写（符合规范）。

二、HTML 基础框架

一个完整的 HTML 文档有固定的基本结构，以下是最基础的框架：

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>网页标题</title>
</head>
<body>
    <!-- 这里是网页的可见内容 -->
    <h1>这是一级标题</h1>
    <p>这是一段文本。</p>
</body>
</html>

三、框架各部分说明

  1. <!DOCTYPE html>
    • 声明文档类型，告诉浏览器这是一个 HTML5 文档（必须放在文档最开头）。
    • 不属于 HTML 标签，仅作为声明。
  2. <html>标签
    • 整个 HTML 文档的根标签，所有内容都包裹在其中。
    • lang="zh-CN" 属性：指定文档主要语言为中文（有助于搜索引擎和辅助工具识别）。
  3. <head> 标签
    • 包含文档的元数据（metadata），即不直接显示在页面上的信息（如标题、编码方式、样式等）。
    • 常见子标签：
      • <meta charset="UTF-8">：指定文档编码为 UTF-8（支持中文等多语言）。
      • <meta name="viewport" ...>：用于响应式设计，确保移动端页面正确缩放。
      • <title>：定义网页标题（显示在浏览器标签页上，是搜索引擎优化的重要元素）。
  4. <body> 标签
    • 包含网页的可见内容，如文本、图片、链接、按钮等，浏览器会将这里的内容渲染到页面上。

四、入门示例

以下是一个简单的完整 HTML 页面示例，可直接保存为 .html 文件（如 index.html），用浏览器打开即可查看效果：

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>我的第一个网页</title>
</head>
<body>
    <h1>欢迎来到我的网页</h1>
    <p>这是我用 HTML 写的第一个页面。</p>
    <a href="https://www.example.com">点击访问示例网站</a>
    <img src="https://picsum.photos/200/100" alt="示例图片">
</body>
</html>

五、<body>子标签

文本与标题标签

用于组织页面文本内容，明确层级关系。

  1. 标题标签（ <h1>到<h6>）表示不同级别的标题，<h1>级别最高 （通常一个页面只有一个）， <h6>最低。

    属性：align（有left | center | right 三个值）。

    <h1>一级标题（页面主标题）</h1>
<h2>二级标题（章节标题）</h2>
<h3>三级标题（小节标题）</h3>
<!-- 以此类推到 h6 -->

<h6 align="left">六级标题 </h6>

  2. 段落标签（<p>）定义一段文本，浏览器会自动在段落前后添加换行和间距。

    <p>这是一段普通文本，用于描述内容。</p>
<p>段落标签会自动分隔内容，让页面更易读。</p>

  3. 换行标签（<br>）强制换行（单标签，无需闭合），常用于段落内的换行。

    <p>第一行文本<br>第二行文本（通过 br 标签换行）</p>

  4. 水平线标签（<hr>）

    <hr color="grean" width="300px" size="20px" align="left | right" />
<!-- 默认居中 -->

  5. 文本格式化标签

    用于强调或修饰文本：

加重语气
定义着重字
删除线文本
定义斜体文本
定义粗体文本

### 链接标签（`<a>`）

定义超链接，用于跳转至其他页面、文件或同一页面的指定位置。

核心属性：`href`（目标地址）`target`（打开方式，`_blank`表示新窗口）。

```html
<!-- 跳转到外部网站 -->
<a href="https://www.baidu.com" target="_blank">百度一下</a>

<!-- 跳转到本地页面（同一文件夹下的 page.html） -->
<a href="page.html">访问内部页面</a>

<!-- 跳转到页面内的指定位置（需配合 id 属性） -->
<a href="#section1">跳至第一部分</a>
<h2 id="section1">第一部分内容</h2>

<!-- 邮件链接（点击打开邮件客户端） -->
<a href="mailto:example@mail.com">发送邮件</a>

图像标签（<img>）

插入图片（单标签），核心属性：

  • src：图片路径（本地路径或网络 URL）。
  • alt：图片加载失败时的替代文本（必填，提升可访问性）。
  • width/height：图片宽度 / 高度（单位：像素或百分比）。
<!-- 插入网络图片 -->
<img src="https://picsum.photos/400/200" alt="示例风景图" width="400">

<!-- 插入本地图片（假设图片在 images 文件夹下 -->
<img src="images/photo.jpg" alt="本地照片" height="200">

<!-- alt为规定图片替代的文本，title为鼠标指向时显示的图片文本 -->
<img src="" width="" height="" alt="" tile"">

列表标签

用于展示有序或无序的列表项。

  1. 无序列表（<ul> + <li>）列表项前默认显示圆点（样式可通过 CSS 修改）。

    <ul>
  <li>苹果</li>
  <li>香蕉</li>
  <li>橙子</li>
</ul>

  2. 有序列表（<ol> + <li>）列表项前显示数字（默认）、字母等序号。

    <ol>
  <li>第一步：打开浏览器</li>
  <li>第二步：输入网址</li>
  <li>第三步：按下回车</li>
</ol>

  3. 定义列表（<dl> + <dt> + <dd>）用于展示 “术语 - 解释” 类内容。

    <dl>
  <dt>HTML</dt>
  <dd>超文本标记语言，用于构建网页结构。</dd>
  <dt>CSS</dt>
  <dd>层叠样式表，用于美化网页。</dd>
</dl>

容器标签

用于布局和分组内容，无默认样式，需配合 CSS 使用。

  1. <div>：块级容器（独占一行），常用于大区块布局（如头部、主体、底部）。

  2. <span>：行内容器（不独占一行），常用于包裹文本中的部分内容（如单独修饰某段文字）。

    <p>我喜欢 <span style="color: red;">红色</span> 和 <span style="color: blue;">蓝色</span>。</p>

表格标签（<table>）

用于展示结构化数据（如成绩单、价格表）。

核心标签：<table>（表格）、<tr>（行）、<td>（单元格）、<th>（表头单元格，默认加粗居中）。

<table border="1">  <!-- border 属性添加边框（仅示例用，实际推荐用 CSS） -->
  <tr>
    <th>姓名</th>
    <th>年龄</th>
  </tr>
  <tr>
    <td>张三</td>
    <td>20</td>
  </tr>
  <tr>
    <td>李四</td>
    <td>22</td>
  </tr>
</table>


<table>
    <tr>
        <td colspan="2">单元格1和2</td>
        <td>单元格3</td>
        <td>单元格4</td>
    </tr>
    
    <tr>
        <td>单元格5</td>
        <td>单元格6</td>
        <td>单元格7</td>
        <td rowspan="2">单元格8和12</td>
    </tr>
    
    <tr>
        <td>单元格9</td>
        <td>单元格10</td>
        <td>单元格11</td>
    </tr>
    
</table>

表单标签（<form>）

用于收集用户输入（如登录、注册、搜索），配合输入类标签使用。

核心属性：action（提交数据的地址）、method（提交方式、get或post）

常用输入标签：

  • <input>：单行文本、密码、复选框等（通过 type 属性控制类型）。
  • <textarea>：多行文本输入。
  • <select> + <option>：下拉选择框。
  • <button>：提交或重置按钮。
<form action="/submit" method="post">
  <p>
    <label for="name">姓名：</label>
    <input type="text" id="name" name="username">  <!-- name 用于后端接收数据 -->
  </p>
  <p>
    <label for="pwd">密码：</label>
    <input type="password" id="pwd" name="password">
  </p>
  <p>
    <label>性别：</label>
    <input type="radio" name="gender" value="male"> 男
    <input type="radio" name="gender" value="female"> 女
  </p>
  <p>
    <button type="submit">提交</button>
    <button type="reset">重置</button>
  </p>
</form>

这些标签是构建网页内容的基础，实际开发中会结合 CSS（美化样式）和 JavaScript（添加交互），让页面更丰富。
