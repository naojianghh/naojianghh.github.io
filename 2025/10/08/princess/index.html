队列4.2 Rescue the princess

2025/10/8 算法

广度优先搜索

数据结构与算法实验题 4.2 Rescue the princess

★实验任务

有一天，公主被一个魔王抓走关在一个城堡里面，城堡可以用 NM 的矩阵来描述，1<N,M<=200。监狱由 NM 个方格组成，每个方格中可能为墙壁，空地，怪物，公主或者是勇士。

现在勇士想去营救公主。他的任务是找到公主。约定，“找到公主”的意思是到达公主被关的位置。如果勇士想到达某个方格，但方格中有怪物，那么必须杀死怪物，才能到达这个方格。假设勇士只能向上，下，左，右四个方向移动一步。移动一步的需要花费 1 个单位的时间，杀死怪物也需要一个单位的时间。

试着计算勇士找到公主需要多长时间。只能上，下，左，右移动，而且墙壁不能通过。

★数据输入

输入第一行包含 2 个整数 N，M，接下来 N 行，每行 M 个字符：“.”代表空地，“a”表示公主，“r”表示勇士，“#”代表墙壁，“x”表示怪物。（测试数据中“a”和“r”只有一个）。

★数据输出

输出一个整数，表示找到公主所需要最短的时间。如果无法找到公主，输出-1。

输入示例
7 8
#.#####.
#.a#..r.
#..#x…
..#..#.#
#…##..
.#……
……..
输出示例
13

深度优先搜索

#include<iostream>
#include<stack>
#include<string>
#include<vector>
#include<queue>

using namespace std;


int n, m;
int ans = 1996;
queue<pair<int, int>> p;
int nx[4] = { 0,1,0,-1 };
int ny[4] = { 1,0,-1,0 };

void search(char** map,pair<int, int> loc,char count) {
    int x = loc.first;
    int y = loc.second;
    for (int i = 0; i < 4; i++) {
        int dx = x + nx[i];
        int dy = y + ny[i];
        if (map[dx][dy] == '.') {
            map[dx][dy] = count + 1;
            p.push(pair<int,int>(dx, dy));
        }
        else if (map[dx][dy] == 'x') {
            map[dx][dy] = count + 2;
            p.push(pair<int,int>(dx, dy));
        }
        else if (map[dx][dy] == 'a') {
            int temp = (int)(count + 1 - '0');
            if (temp < ans) {
                ans = temp;
            }
            p.pop();
            return;
        }
    }
    p.pop();
    while (!p.empty()) {
        search(map, p.front(), map[p.front().first][p.front().second]);
    }
}

int main()
{
    cin >> n >> m;
    int x, y;
    char** map = new char* [n + 2];
    for (int i = 0; i <= n + 1; i++) {
        map[i] = new char[m + 2];
    }
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= m; j++) {
            cin >> map[i][j];
            if (map[i][j] == 'r') {
                x = i;
                y = j;
            }
        }
    }
    for (int i = 0; i < m + 2; i++) {
        map[0][i] = '#';
        map[n + 1][i] = '#';
    }
    for (int i = 0; i < n + 2; i++) {
        map[i][0] = '#';
        map[i][m + 1] = '#';
    }
    pair <int, int>temp = {x,y};
    p.push(temp);
    search(map, temp, '0');
    if (ans != 1996) {
        cout << ans;
    }
    else{
        cout << -1;
    }
    return 0;
}

自己凭感觉写的，虽然用了队列，但是更接近深度优先搜索。

核心思路：尝试通过 “队列辅助的递归搜索” 从起点'r'找到终点'a'，并计算路径长度，本质上是想实现 “最短路径搜索”，但因逻辑设计问题实际偏向深度优先搜索（DFS）。具体可拆解为以下几点：

  1. 初始设定
    • 用二维数组map存储迷宫，'r'是起点，'a'是终点，'.'和'x'是可通行区域（可能'x'被视为特殊可走节点），'#'是边界（不可通行）。
    • 用队列p存储待探索的坐标，初始将起点'r'的坐标入队。
  2. 搜索逻辑
    • 定义search函数，参数为当前地图、当前坐标loc、当前路径长度count（用字符表示，如'0'代表 0 步）。
    • 对当前坐标的 4 个方向（上下左右）进行探索：
      • 若遇到'.'（普通可走），标记为count + 1（更新路径长度），并将该坐标入队。
      • 若遇到'x'（特殊可走），标记为count + 2（可能想区分路径类型），并将该坐标入队。
      • 若遇到'a'（终点），计算当前路径长度并更新最短路径ans，然后退出。
  3. 队列与递归的结合
    • 每次处理完当前节点后，弹出队列头部元素，再通过递归依次处理队列中剩余的节点，试图遍历所有可能的路径。

广度优先搜索

#include<iostream>
#include<stack>
#include<string>
#include<vector>
#include<queue>

using namespace std;


int n, m;
int ans = 1996;
queue<pair<int, int>> p;
int nx[4] = { 0,1,0,-1 };
int ny[4] = { 1,0,-1,0 };

void search(char** map,pair<int, int> loc,char count) {
    while (!p.empty()) {
        int x = p.front().first;
        int y = p.front().second;
        for (int i = 0; i < 4; i++) {
            int dx = x + nx[i];
            int dy = y + ny[i];
            if (map[dx][dy] == '.') {
                map[dx][dy] = map[x][y] + 1;
                p.push(pair<int, int>(dx, dy));
            }
            else if (map[dx][dy] == 'x') {
                map[dx][dy] = map[x][y] + 2;
                p.push(pair<int, int>(dx, dy));
            }
            else if (map[dx][dy] == 'a') {
                int temp = (int)(map[x][y] + 1 - '0');
                if (temp < ans) {
                    ans = temp;
                }
            }
            else if (map[dx][dy] != '#') {
                if (map[x][y] + 1 < map[dx][dy]) {
                    map[dx][dy] = map[x][y] + 1;
                }
            }
        }
        p.pop();
    }
   
}

int main()
{
    cin >> n >> m;
    int x, y;
    char** map = new char* [n + 2];
    for (int i = 0; i <= n + 1; i++) {
        map[i] = new char[m + 2];
    }
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= m; j++) {
            cin >> map[i][j];
            if (map[i][j] == 'r') {
                map[i][j] = '0';
                x = i;
                y = j;
            }
        }
    }
    for (int i = 0; i < m + 2; i++) {
        map[0][i] = '#';
        map[n + 1][i] = '#';
    }
    for (int i = 0; i < n + 2; i++) {
        map[i][0] = '#';
        map[i][m + 1] = '#';
    }
    pair <int, int>temp = {x,y};
    p.push(temp);
    search(map, temp, '0');
    if (ans != 1996) {
        cout << ans;
    }
    else{
        cout << -1;
    }
    return 0;
}

这个代码的核心思路是 用队列实现 BFS 框架，从起点'r'出发探索迷宫，寻找终点'a'，并尝试让节点保留更小的路径步数，具体分 4 步：

  1. 初始化与准备

    • 地图外围填'#'当边界，避免越界；起点'r'改成字符'0'（作为路径长度的起始基准），并把起点坐标加入队列。
    • 用队列p存储待探索的节点，按 “先进先出” 顺序处理，符合 BFS “逐层扩散” 的特点。

  2. BFS 核心探索

    • 循环从队列取节点

      (x,y)

      ，遍历它的上下左右 4 个邻居

      (dx,dy)

      ，根据邻居类型处理：

      • 若邻居是'.'（普通可走）：把邻居标记为 “当前节点步数 + 1”（字符累加，比如'0'→'1'），并加入队列；
      • 若邻居是'x'（特殊可走）：把邻居标记为 “当前节点步数 + 2”（给'x'更高的路径成本），也加入队列；
      • 若邻居是'a'（终点）：计算当前节点到终点的步数，更新全局最小步数ans。

  3. 新增的 “小步数更新” 逻辑

    • 当邻居既不是'.','x','a','#'

      （即已被标记为数字的节点）时，判断 “当前节点的步数 + 1” 是否比邻居现有步数小：

      • 若是，就把邻居的步数更新为更小的值（试图修正之前可能因'x'导致的大步数）。

  4. 结果判断

    • 探索结束后，若ans被更新（不是初始的 1996），输出最小步数；否则输出-1（没找到终点）。
