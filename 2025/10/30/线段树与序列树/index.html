线段树与序列树详解

线段树（Segment Tree）

一、基本概念

线段树是一种二叉树数据结构，主要用于解决区间查询和区间更新问题。

二、核心特性

  • 适用场景：区间求和、区间最值、区间更新等
  • 时间复杂度：
    • 建树：O(n)
    • 区间查询：O(log n)
    • 单点更新：O(log n)
    • 区间更新：O(log n)（使用懒惰传播）
  • 空间复杂度：O(4n)（数组实现）

三、数据结构定义

class SegmentTree {
private:
    vector<int> tree;  // 线段树数组
    vector<int> arr;   // 原始数组
    vector<int> lazy;  // 懒惰标记数组（用于区间更新）
    int n;
    
public:
    SegmentTree(vector<int>& nums) {
        n = nums.size();
        arr = nums;
        tree.resize(4 * n);
        lazy.resize(4 * n, 0);
        build(0, n-1, 0);
    }
};

四、建树过程

// 递归建树
void build(int start, int end, int node) {
    if (start == end) {
        // 叶子节点，存储单个元素
        tree[node] = arr[start];
        return;
    }
    
    int mid = (start + end) / 2;
    int leftNode = 2 * node + 1;
    int rightNode = 2 * node + 2;
    
    // 递归构建左右子树
    build(start, mid, leftNode);
    build(mid + 1, end, rightNode);
    
    // 合并左右子树结果（这里是求和）
    tree[node] = tree[leftNode] + tree[rightNode];
}

五、区间查询

// 查询区间 [l, r] 的和
int query(int l, int r, int start, int end, int node) {
    if (l <= start && end <= r) {
        // 当前区间完全在查询范围内
        return tree[node];
    }
    
    int mid = (start + end) / 2;
    int leftNode = 2 * node + 1;
    int rightNode = 2 * node + 2;
    int sum = 0;
    
    if (l <= mid) {
        sum += query(l, r, start, mid, leftNode);
    }
    if (r > mid) {
        sum += query(l, r, mid + 1, end, rightNode);
    }
    
    return sum;
}

// 对外接口
int queryRange(int l, int r) {
    return query(l, r, 0, n-1, 0);
}

六、单点更新

// 更新单个元素
void update(int index, int value, int start, int end, int node) {
    if (start == end) {
        // 找到目标叶子节点
        arr[index] = value;
        tree[node] = value;
        return;
    }
    
    int mid = (start + end) / 2;
    int leftNode = 2 * node + 1;
    int rightNode = 2 * node + 2;
    
    if (index <= mid) {
        update(index, value, start, mid, leftNode);
    } else {
        update(index, value, mid + 1, end, rightNode);
    }
    
    // 更新父节点
    tree[node] = tree[leftNode] + tree[rightNode];
}

七、懒惰传播（区间更新）

// 懒惰传播更新
void updateLazy(int l, int r, int value, int start, int end, int node) {
    // 先处理懒惰标记
    if (lazy[node] != 0) {
        tree[node] += (end - start + 1) * lazy[node];
        if (start != end) {
            // 将懒惰标记传递给子节点
            lazy[2*node+1] += lazy[node];
            lazy[2*node+2] += lazy[node];
        }
        lazy[node] = 0;
    }
    
    if (start > r || end < l) return; // 完全不重叠
    
    if (l <= start && end <= r) {
        // 当前区间完全在更新范围内
        tree[node] += (end - start + 1) * value;
        if (start != end) {
            lazy[2*node+1] += value;
            lazy[2*node+2] += value;
        }
        return;
    }
    
    int mid = (start + end) / 2;
    updateLazy(l, r, value, start, mid, 2*node+1);
    updateLazy(l, r, value, mid+1, end, 2*node+2);
    tree[node] = tree[2*node+1] + tree[2*node+2];
}

序列树（树状数组）

一、基本概念

1.1 什么是序列树？

序列树（Fenwick Tree / Binary Indexed Tree） 是一种用于高效处理前缀和查询和单点更新的数据结构。

1.2 核心思想

利用二进制表示的特性，将前缀和分解为多个子区间的和，从而实现：

  • 前缀和查询：O(log n)
  • 单点更新：O(log n)

二、核心原理 - Lowbit操作

2.1 Lowbit定义

**Lowbit(x)**：获取x的二进制表示中最低位的1所代表的数值

int lowbit(int x) {
    return x & (-x);
}

2.2 Lowbit示例

x = 6  (二进制: 110)  → lowbit(6) = 2  (二进制: 010)
x = 12 (二进制: 1100) → lowbit(12) = 4 (二进制: 0100)
x = 7  (二进制: 111)  → lowbit(7) = 1  (二进制: 001)

三、序列树的结构原理

3.1 基本结构

序列树不是一棵显式的树，而是通过数组和下标关系隐含的树形结构。

数组下标从1开始（重要！）

  • tree[i]：存储区间[i - lowbit(i) + 1, i]的和

3.2 存储区间示意图

对于长度为8的数组：

tree[1] = arr[1]                                          [1,1]
tree[2] = arr[1] + arr[2]                                 [1,2]  
tree[3] = arr[3]                                          [3,3]
tree[4] = arr[1] + arr[2] + arr[3] + arr[4]               [1,4]
tree[5] = arr[5]                                          [5,5]
tree[6] = arr[5] + arr[6]                                 [5,6]
tree[7] = arr[7]                                          [7,7]
tree[8] = arr[1] + arr[2] + ... + arr[8]                  [1,8]

3.3 树形结构关系

                tree[8]
               /       \
          tree[4]     tree[6]   tree[7]
          /     \     /     
     tree[2] tree[3] tree[5] 
     /
tree[1]

四、关键操作原理

4.1 单点更新原理

目标：更新arr[i]时，更新所有包含arr[i]的tree节点

更新路径：沿着二进制下标不断加上lowbit(i)

  • 从i开始
  • i = i + lowbit(i)
  • 重复直到i > n
void update(int index, int value) {
    int i = index + 1;  // 序列树下标从1开始
    while (i <= n) {
        tree[i] += value;
        i += lowbit(i);  // 跳到父节点
    }
}

更新示例：更新arr[3]（即tree[4]中的元素）

更新路径：3 → 4 → 8

更新arr[3] = 5:

1. tree[3] += 5    (包含arr[3])
2. tree[4] += 5    (包含arr[1]-arr[4])  
3. tree[8] += 5    (包含arr[1]-arr[8])

4.2 前缀和查询原理

目标：计算arr[1] + arr[2] + ... + arr[i]的和

查询路径：沿着二进制下标不断减去lowbit(i)

  • 从i开始
  • i = i - lowbit(i)
  • 重复直到i == 0
int query(int index) {
    int sum = 0;
    int i = index + 1;  // 序列树下标从1开始
    while (i > 0) {
        sum += tree[i];
        i -= lowbit(i);  // 跳到前一个区间
    }
    return sum;
}

查询示例：查询前缀和[1,7]

查询路径：7 → 6 → 4 → 结束

查询前缀和[1,7]:
sum = tree[7] + tree[6] + tree[4]

tree[7] = arr[7]                    [7,7]
tree[6] = arr[5] + arr[6]           [5,6]  
tree[4] = arr[1] + arr[2] + arr[3] + arr[4]  [1,4]

总和 = arr[1]到arr[7]的和

五、数学原理深度解析

5.1 二进制分解原理

关键观察：任何数字都可以表示为2的幂次之和

7 = 4 + 2 + 1
13 = 8 + 4 + 1

序列树利用这个性质将前缀和分解：

sum[1,7] = sum[1,4] + sum[5,6] + sum[7,7]
         = tree[4] + tree[6] + tree[7]

5.2 区间覆盖证明

定理：对于任意i，区间[i-lowbit(i)+1, i]的长度正好是lowbit(i)

证明：

  • 设i的二进制为：xxxx100...0（最后k个0）
  • lowbit(i) = 2^k
  • i - lowbit(i) = xxxx000...0
  • 区间长度 = i - (i - lowbit(i)) = lowbit(i)

5.3 操作正确性证明

更新正确性：每个被更新的节点确实包含目标元素

  • 因为更新路径是不断加上lowbit，覆盖的区间越来越大
  • 每个父节点都包含子节点的区间

查询正确性：所有子区间正好覆盖[1, i]

  • 每次减去lowbit，得到不相交的区间
  • 这些区间的并集正好是[1, i]

六、完整实现与示例

6.1 完整序列树实现

#include <iostream>
#include <vector>
using namespace std;

class FenwickTree {
private:
    vector<int> tree;
    int n;
    
    int lowbit(int x) {
        return x & (-x);
    }
    
public:
    // 构造函数：从数组构建
    FenwickTree(vector<int>& nums) {
        n = nums.size();
        tree.resize(n + 1, 0);
        for (int i = 0; i < n; i++) {
            update(i, nums[i]);
        }
    }
    
    // 单点更新：在位置index增加value
    void update(int index, int value) {
        int i = index + 1;  // 转为1-based索引
        while (i <= n) {
            tree[i] += value;
            i += lowbit(i);
        }
    }
    
    // 查询前缀和：[0, index]的和
    int query(int index) {
        int sum = 0;
        int i = index + 1;  // 转为1-based索引
        while (i > 0) {
            sum += tree[i];
            i -= lowbit(i);
        }
        return sum;
    }
    
    // 查询区间和：[l, r]的和
    int queryRange(int l, int r) {
        if (l == 0) return query(r);
        return query(r) - query(l - 1);
    }
    
    // 打印树结构（用于调试）
    void printTree() {
        cout << "序列树结构:" << endl;
        for (int i = 1; i <= n; i++) {
            cout << "tree[" << i << "] = " << tree[i] 
                 << " (覆盖区间: [" << i - lowbit(i) + 1 << ", " << i << "])" << endl;
        }
    }
};

6.2 运行示例

int main() {
    vector<int> arr = {1, 3, 5, 7, 9, 11};
    FenwickTree ft(arr);
    
    ft.printTree();
    
    cout << "\n前缀和查询:" << endl;
    cout << "前缀和[0,2]: " << ft.query(2) << " (应为: 1+3+5=9)" << endl;
    cout << "前缀和[0,4]: " << ft.query(4) << " (应为: 1+3+5+7+9=25)" << endl;
    
    cout << "\n区间和查询:" << endl;
    cout << "区间[2,4]: " << ft.queryRange(2, 4) << " (应为: 5+7+9=21)" << endl;
    
    cout << "\n更新测试:" << endl;
    ft.update(3, 10);  // arr[3]从7增加到17
    cout << "更新后区间[2,4]: " << ft.queryRange(2, 4) << " (应为: 5+17+9=31)" << endl;
    
    return 0;
}
