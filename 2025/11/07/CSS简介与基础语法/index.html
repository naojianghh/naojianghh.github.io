CSS简介与基础语法

2025/11/7 web前端

CSS（层叠样式表）的核心作用是为 HTML 元素设置样式，其语法规则简洁清晰，核心由「选择器」「声明块」「属性 - 值对」三部分构成，同时包含注释、引入方式等基础规范。

CSS

一、核心语法结构

CSS 规则的基本格式如下，核心是「选中元素 + 定义样式」：

/* 选择器：指定要样式化的 HTML 元素 */
选择器 {
  /* 声明块：包裹一组样式声明（属性-值对） */
  属性1: 值1; /* 样式声明：属性和值用冒号分隔，结尾必须加分号（可省略最后一个，但推荐都加） */
  属性2: 值2;
}

1. 关键组成部分说明

组成部分 作用 示例
选择器 定位 HTML 中需要样式化的元素（如标签、类、ID 等） p（选中所有 `` 标签）、.box（选中所有 class="box" 的元素）
声明块 用 {} 包裹，包含一个或多个样式声明 { color: red; font-size: 16px; }
属性 CSS 预定义的样式名（如颜色、字体、宽度等） color（文字颜色）、width（宽度）、background（背景）
值 对应属性的具体设置（需符合属性的取值规范） red（颜色值）、16px（尺寸值）、center（对齐方式）

2. 基础示例

/* 选中所有 <h1> 标签，设置文字颜色红色、字体大小 24px */
h1 {
  color: red;
  font-size: 24px;
}

/* 选中 class="container" 的元素，设置宽度 1200px、水平居中 */
.container {
  width: 1200px;
  margin: 0 auto;
}

二、选择器：如何选中元素？

选择器是 CSS 的核心，用于精准定位 HTML 元素，以下是最常用的基础选择器：

选择器类型 语法 作用 示例
元素选择器 标签名 选中所有该标签的元素 div { ... }（选中所有 <div>）
类选择器 .类名 选中所有 class="类名" 的元素（可复用） .active { color: blue; }（选中 <p class="active">）
ID 选择器 #ID名 选中唯一 id="ID名" 的元素（不可重复） #header { height: 80px; }（选中 <header id="header">）
通配符选择器 * 选中页面所有元素（慎用，性能较低） * { margin: 0; padding: 0; }（清除所有元素默认边距）
后代选择器 选择器 1 选择器 2 选中选择器 1 内部的所有选择器 2 元素 nav a { text-decoration: none; }（选中 <nav> 内所有 <a>）
交集选择器 选择器 1 选择器 2 同时满足两个选择器的元素（无空格） p.active { font-weight: bold; }（选中 <p class="active">）
并集选择器 选择器 1, 选择器 2 同时选中多个选择器的元素（用逗号分隔） h1, h2, h3 { color: #333; }（选中所有 h1/h2/h3）

三、样式声明：属性与值的规则

1. 属性值的常见类型

CSS 属性值需符合对应属性的规范，常见类型：

  • 颜色值：red（英文）、#ff0000（十六进制）、rgb(255,0,0)（RGB）、rgba(255,0,0,0.5)（带透明度）
  • 尺寸值：10px（像素，固定尺寸）、2em（相对当前字体大小）、50%（相对父元素尺寸）、auto（自动）
  • 关键字：center（居中）、block（块级）、none（无）、bold（加粗）

2. 声明的注意事项

  • 分号必须：每个声明结尾必须加 ;，否则后续样式可能失效（最后一个声明可省略，但推荐统一添加，避免出错）
  • 大小写不敏感：CSS 对大小写不敏感（如 Color: Red 等价于 color: red），但建议统一小写（规范）
  • 空格不影响：声明中可适当加空格（如 font-size: 16px 等价于 font-size:16px），增强可读性

四、注释：代码说明（不生效）

CSS 注释用于解释代码，浏览器会忽略注释内容，语法：

/* 单行注释 */
p {
  color: #333; /* 文字颜色：深灰色（行内注释） */
}

/*
  多行注释
  选中 .box 元素，设置背景色和内边距
*/
.box {
  background: #f5f5f5;
  padding: 20px;
}

注意：CSS 没有 // 注释语法，// 会导致后续代码失效！

五、CSS 的引入方式（3 种核心）

CSS 需引入 HTML 才能生效，3 种常用方式：

1. 内联样式（行内样式）

直接写在 HTML 元素的 style 属性中，优先级最高（仅作用于当前元素）：

<p style="color: green; font-size: 18px;">这是内联样式的文字</p>

缺点：样式与结构耦合，复用性差，仅适合临时修改单个元素。

2. 内部样式表（嵌入式）

写在 HTML 的 标签中（通常放在 内），作用于当前 HTML 文件：

<!DOCTYPE html>
<html>
<head>
  <style>
    /* 内部样式表 */
    h1 { color: purple; }
    .content { line-height: 1.5; }
  </style>
</head>
<body>
  <h1>标题</h1>
  <div class="content">内容</div>
</body>
</html>

优点：样式与结构分离（同一文件内），适合单页面样式。

3. 外部样式表（链接式）

将 CSS 写在独立的 .css 文件中，通过 `` 标签引入 HTML，最推荐（复用性强、维护方便）：

步骤 1：创建 CSS 文件（如 style.css）

/* style.css */
body { background: #f9f9f9; }
a { color: #0066cc; text-decoration: none; }

步骤 2：HTML 中引入

<!DOCTYPE html>
<html>
<head>
  <!-- 引入外部 CSS 文件 -->
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <a href="#">这是链接</a>
</body>
</html>

优点：样式与结构完全分离，多个页面可共用一个 CSS 文件，便于统一维护。

六、基础语法总结

  1. 核心结构：选择器 { 属性: 值; }
  2. 选择器：优先掌握「类选择器」「元素选择器」「后代选择器」（最常用）
  3. 声明：属性和值需匹配，分号不能漏，注释用 /* */
  4. 引入方式：推荐「外部样式表」，少量样式可用「内部样式表」，避免滥用「内联样式」
