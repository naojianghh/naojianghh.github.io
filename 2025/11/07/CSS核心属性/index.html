CSS核心属性

2025/11/7 web CSS

开发中最常用的 字体、背景、文本、表格 属性，包含语法、取值、示例，兼顾实用性和易理解性，直接可复制到项目中使用。

CSS 核心属性详解（字体、背景、文本、表格）

一、字体属性（控制文字样式）

核心用于设置文字的 字体、大小、粗细、风格、行高 等，是页面文字排版的基础。

属性名 作用 取值 / 说明 示例
font-family 设置字体（优先匹配） 具体字体名（如 Arial）、通用字体族（如 sans-serif 无衬线、serif 衬线）；多个字体用逗号分隔（fallback 机制） font-family: "微软雅黑", "Microsoft YaHei", sans-serif;
font-size 设置字体大小（核心） 固定值（16px 常用）、相对值（1.2em 相对于父元素、120% 百分比、small/large 关键字） font-size: 16px;（网页默认 16px）
font-weight 设置字体粗细 关键字（normal 正常 /bold 加粗 /bolder 更粗 /lighter 更细）、数字（100-900，400=normal，700=bold） font-weight: 600;（半加粗，常用）
font-style 设置字体风格（斜体 / 正常） normal（正常）、italic（斜体，字体自带）、oblique（强制斜体） font-style: italic;（斜体文字）
line-height 设置行高（文字垂直间距） 数字（如 1.5，相对于字体大小的倍数，推荐）、固定值（24px）、百分比（150%） line-height: 1.5;（行高 = 字体大小 ×1.5，易读）
font 复合属性（简化写法） 顺序：font-style font-weight font-size/line-height font-family（必须包含 font-size 和 font-family） font: italic 600 16px/1.5 "微软雅黑", sans-serif;
color 设置文字颜色（关联字体） 颜色值（red、#333、rgb(51,51,51)、rgba(51,51,51,0.8) 带透明度） color: #666;（浅灰色文字）

字体属性常用示例

.text {
  font-family: "PingFang SC", "微软雅黑", sans-serif; /* 适配 macOS/Windows */
  font-size: 14px;
  font-weight: normal; /* 取消加粗 */
  line-height: 1.6; /* 优化可读性 */
  color: rgba(33, 33, 33, 0.9); /* 深灰带轻微透明度 */
}

二、背景属性（控制元素背景）

用于设置元素的 背景色、背景图、背景平铺、位置、大小 等，丰富元素视觉效果。

属性名 作用 取值 / 说明 示例
background-color 设置背景颜色 颜色值、transparent（透明，默认） background-color: #f5f5f5;（浅灰背景）
background-image 设置背景图片 url("图片路径")（相对 / 绝对路径）、none（无背景图，默认） background-image: url("bg.jpg");
background-repeat 控制背景图平铺方式 repeat（默认，全平铺）、no-repeat（不平铺）、repeat-x（水平平铺）、repeat-y（垂直平铺） background-repeat: no-repeat;
background-position 控制背景图位置 关键字（center 居中、left top 左上）、数值（50px 30px 左 50px 上 30px）、百分比（50% 50% 居中） background-position: center;
background-size 控制背景图大小 cover（覆盖元素，可能裁剪）、contain（完全显示，可能留白）、数值（100px 80px 宽高）、百分比（100% 100% 拉伸填充） background-size: cover;（常用背景铺满）
background-attachment 背景图滚动行为 scroll（随页面滚动，默认）、fixed（固定不动，类似视差效果） background-attachment: fixed;
background 复合属性（简化写法） 顺序：background-color background-image background-repeat background-position / background-size background-attachment（size 前必须加 /） background: #f5f5f5 url("bg.jpg") no-repeat center / cover fixed;

背景属性常用示例（全屏背景图）

body {
  background: url("full-screen-bg.jpg") no-repeat center / cover;
  background-color: #eee; /* 背景图加载失败时的 fallback 色 */
}

三、文本属性（控制文本排版）

用于设置文本的 对齐方式、缩进、间距、装饰线 等，优化文本布局的可读性。

属性名 作用 取值 / 说明 示例
text-align 文本水平对齐 left（左对齐，默认）、center（居中）、right（右对齐）、justify（两端对齐，仅多行有效） text-align: center;（标题居中）
text-indent 文本首行缩进 数值（2em 常用，对应 2 个字符宽度）、百分比（10%） text-indent: 2em;（段落首行缩进 2 字）
letter-spacing 字符间距（字与字之间） 数值（2px 加宽、-1px 收紧）、normal（默认） letter-spacing: 1px;（加宽字符间距）
word-spacing 单词间距（英文单词之间） 数值（5px 加宽）、normal（默认）；中文无效（中文无单词概念） word-spacing: 3px;（英文单词间距）
text-decoration 文本装饰线（下划线 / 删除线等） none（无装饰，默认，常用於清除 <a> 标签下划线）、underline（下划线）、line-through（删除线）、overline（上划线）；可加颜色（如 underline red） text-decoration: none;（清除 a 标签下划线）
text-transform 英文文本大小写转换 none（默认）、uppercase（全大写）、lowercase（全小写）、capitalize（首字母大写） text-transform: uppercase;（英文全大写）
white-space 空白字符处理（换行 / 空格） normal（默认，合并空格 + 自动换行）、nowrap（不换行，溢出隐藏或滚动）、pre（保留空格和换行，类似 <pre> 标签） white-space: nowrap;（强制不换行）
overflow-wrap 长文本换行（防止溢出） normal（默认）、break-word（单词内换行，适合中文 / 长英文单词） overflow-wrap: break-word;（避免长文本撑破容器）

文本属性常用示例（段落排版）

.paragraph {
  text-align: justify; /* 两端对齐 */
  text-indent: 2em; /* 首行缩进2字 */
  letter-spacing: 0.5px; /* 字符间距轻微加宽 */
  line-height: 1.8; /* 行高优化可读性 */
  overflow-wrap: break-word; /* 长文本换行 */
}

a {
  text-decoration: none; /* 清除下划线 */
  color: #0066cc;
}
a:hover {
  text-decoration: underline; /* hover 时显示下划线 */
}

四、表格属性（控制表格样式）

用于设置表格的 边框、间距、对齐、单元格样式 等，默认表格样式简陋，需通过 CSS 美化。

属性名 作用 取值 / 说明 示例
border-collapse 表格边框合并（核心） collapse（边框合并，推荐）、separate（边框分离，默认，有间距） border-collapse: collapse;（消除边框间隙）
border-spacing 表格边框间距（仅 separate 时有效） 数值（5px 间距）、0（无间距） border-spacing: 2px;
table-layout 表格布局算法 auto（默认，自适应内容宽度）、fixed（固定列宽，性能更好，需设置 width） table-layout: fixed;（固定表格宽度）
width/height 表格 / 单元格宽高 数值（600px）、百分比（100% 自适应父容器） width: 100%;（表格占满父容器）
text-align 单元格文本水平对齐 继承文本属性的 left/center/right/justify td { text-align: center; }（单元格文本居中）
vertical-align 单元格文本垂直对齐（核心） middle（垂直居中，常用）、top（顶部对齐）、bottom（底部对齐）、baseline（基线对齐） td { vertical-align: middle; }（单元格垂直居中）
border 表格 / 单元格边框 复合属性（宽度 样式 颜色），如 1px solid #ddd（1px 灰色实线） table, td, th { border: 1px solid #ddd; }
empty-cells 空单元格是否显示边框 show（显示，默认）、hide（隐藏空单元格边框） empty-cells: hide;

表格属性常用示例（美化表格）

<!-- HTML 结构 -->
<table class="beautiful-table">
  <thead>
    <tr><th>姓名</th><th>年龄</th><th>城市</th></tr>
  </thead>
  <tbody>
    <tr><td>张三</td><td>25</td><td>北京</td></tr>
    <tr><td>李四</td><td>30</td><td>上海</td></tr>
  </tbody>
</table>

/* CSS 样式 */
.beautiful-table {
  width: 100%; /* 占满父容器 */
  border-collapse: collapse; /* 边框合并 */
  table-layout: fixed; /* 固定列宽 */
  font-size: 14px;
}

.beautiful-table th,
.beautiful-table td {
  border: 1px solid #e8e8e8; /* 边框颜色 */
  padding: 12px 8px; /* 单元格内边距 */
  text-align: center; /* 水平居中 */
  vertical-align: middle; /* 垂直居中 */
}

.beautiful-table th {
  background-color: #f8f8f8; /* 表头背景色 */
  font-weight: 600; /* 表头文字加粗 */
  color: #333;
}

/* 奇偶行交替颜色（斑马纹） */
.beautiful-table tbody tr:nth-child(even) {
  background-color: #fafafa;
}
