并查集9.9_道路

2025/11/25 算法应用 c++ 并查集

并查集的应用

算法与数据结构实验题 9.9 道路

★实验任务

某省调查城镇交通状况，得到现有城镇道路统计表，现给出每条道路连接的城镇编号，问当前的道路设计方案是否合理。

合理的方案为任意两个城镇之间可以相互到达，有且只有一条通路。

★数据输入

输入a b(1<=a,b<=10,000)，表示城镇 a 和城镇 b 连通。输入包含多组数据，每组数据以 0 0 结束。

整个文件以 -1 -1 结尾

★数据输出

如果方案合法，输出”Yes”,否则输出”No”。

输入示例

6 8 5 3 5 2 6 4 5 6 0 0
8 1 7 3 6 2 8 9 7 5 7 4 7 8 7 6 0 0
3 8 6 8 6 4 5 3 5 6 5 2 0 0
1 2 3 4 0 0
-1 -1

输出示例

Yes
Yes
No
No

实现代码

#include <iostream>
#include <vector>
#include <set>
using namespace std;

vector<int> parent;

int find(int x) {
    if (parent[x] != x) {
        parent[x] = find(parent[x]);
    }
    return parent[x];
}

bool unite(int a, int b) {
    int ra = find(a);
    int rb = find(b);
    if (ra == rb) return false;
    parent[ra] = rb;
    return true;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    parent.resize(10001);

    while (true) {
        for (int i = 1; i <= 10000; i++) {
            parent[i] = i;
        }

        int a, b;
        bool valid = true;
        set<int> nodes;

        while (cin >> a >> b) {
            if (a == 0 && b == 0) break;
            if (a == -1 && b == -1) return 0;

            nodes.insert(a);
            nodes.insert(b);

            if (!unite(a, b)) {
                valid = false;
            }
        }
        if (valid && !nodes.empty()) {
            int root = find(*nodes.begin());
            for (int node : nodes) {
                if (find(node) != root) {
                    valid = false;
                    break;
                }
            }
        }

        cout << (valid ? "Yes" : "No") << "\n";
    }

    return 0;
}

核心思路

  1. 利用并查集来判断是否有回路和判断最后道路是否能连通。
  2. 首先初始化各个城镇，然后逐个输入连通的城镇，用unite和并两个城镇集合。
  3. 在合并过程中，用find判断两个集合是否为同一集合（根相同），若为同一集合说明城镇道路有回路，用valid记录下不合理，直接输出答案。
  4. 如果合并完各个城镇并且没有回路，那么开始判断各个城镇是否连通，用nodes记录下参与连通的城镇，遍历nodes记录下的城镇，用find找到各个城镇的集合，判断最终是否只有一个集合（即每个城镇连通），用valid记录下结果，最后输出答案
