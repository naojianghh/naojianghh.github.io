DAG

2025/12/5 算法 c++ 图

有向无环图（Directed Acyclic Graph，简称 DAG）是图论中的核心概念之一，具有 “有向边” 且 “无环路” 的特性，广泛应用于依赖关系建模、调度优化、数据处理等领域。

DAG

一、核心定义与性质

1. 基本定义

  • 有向图：边带有方向的图，每条边用有序对 `` 表示（从顶点 u 指向 v）。
  • 环：存在一条从某个顶点出发，经过若干条边后回到自身的路径（如 u → v → w → u）。
  • DAG：不存在环的有向图。

2. 关键性质

  • 拓扑排序存在性：DAG 一定可以进行拓扑排序（即顶点可以按依赖关系线性排列，所有边的起点在终点之前）。
  • 最短 / 最长路径可解：DAG 中可以用线性时间（O(n+m)，n 为顶点数，m 为边数）求解单源最短 / 最长路径（无需考虑负权环问题）。
  • 强连通分量性质：DAG 的强连通分量都是单个顶点（因为若存在两个顶点强连通，则构成环）。

二、DAG 的核心算法

1. 拓扑排序（Topological Sorting）

拓扑排序是 DAG 的标志性算法，用于将顶点按依赖关系排序，确保所有边的方向与排序顺序一致。

算法实现（两种经典方法）

  • Kahn 算法（入度表 + BFS）：
    1. 计算所有顶点的入度。
    2. 将入度为 0 的顶点加入队列。
    3. 每次取出队首顶点，输出并遍历其邻接顶点，将邻接顶点的入度减 1；若入度变为 0，加入队列。
    4. 重复步骤 3，直到队列为空。若输出的顶点数等于总顶点数，则排序成功；否则图中存在环。
  • DFS 算法（逆后序遍历）：
    1. 对每个未访问的顶点进行 DFS，记录 “后序遍历” 序列（即递归返回时记录顶点）。
    2. 将后序遍历序列反转，得到拓扑排序结果。

示例代码（Kahn 算法，C++）

vector<int> topologicalSort(int n, vector<vector<int>>& graph) {
    vector<int> inDegree(n, 0);
    for (int u = 0; u < n; u++) {
        for (int v : graph[u]) {
            inDegree[v]++;
        }
    }

    queue<int> q;
    for (int i = 0; i < n; i++) {
        if (inDegree[i] == 0) {
            q.push(i);
        }
    }

    vector<int> topo;
    while (!q.empty()) {
        int u = q.front();
        q.pop();
        topo.push_back(u);
        for (int v : graph[u]) {
            if (--inDegree[v] == 0) {
                q.push(v);
            }
        }
    }

    return topo.size() == n ? topo : vector<int>(); // 空向量表示存在环
}

2. DAG 的单源最短路径（Dijkstra 的简化版）

由于 DAG 无环，可先拓扑排序，再按拓扑顺序松弛边，线性时间求解最短路径。

算法步骤

  1. 对 DAG 进行拓扑排序。
  2. 初始化距离数组 dist：起点 dist[s] = 0，其余为无穷大。
  3. 按拓扑顺序遍历每个顶点 u，对每条边 <u, v, w>，执行松弛操作：dist[v] = min(dist[v], dist[u] + w)。
vector<int> shortestPathDAG(int n, vector<vector<pair<int, int>>>& graph, int s) {
    // 1. 拓扑排序（复用Kahn算法）
    vector<int> topo = topologicalSort(n, graph);
    if (topo.empty()) return {}; // 图有环，无法处理

    // 2. 初始化距离数组
    vector<int> dist(n, INT_MAX);
    dist[s] = 0;

    // 3. 按拓扑顺序松弛边
    for (int u : topo) {
        if (dist[u] == INT_MAX) continue; // 不可达
        for (auto [v, w] : graph[u]) {
            if (dist[v] > dist[u] + w) {
                dist[v] = dist[u] + w;
            }
        }
    }

    return dist;
}

3. DAG 的单源最长路径

与最短路径类似，只需将松弛操作改为取最大值，且初始距离设为负无穷大。

算法步骤

  1. 对 DAG 进行拓扑排序。
  2. 初始化距离数组 dist：起点 dist[s] = 0，其余为负无穷大。
  3. 按拓扑顺序遍历每个顶点 u，对每条边 <u, v, w>，执行松弛操作：dist[v] = max(dist[v], dist[u] + w)。

示例代码（C++）

vector<int> longestPathDAG(int n, vector<vector<pair<int, int>>>& graph, int s) {
    vector<int> topo = topologicalSort(n, graph);
    if (topo.empty()) return {};

    vector<int> dist(n, INT_MIN);
    dist[s] = 0;

    for (int u : topo) {
        if (dist[u] == INT_MIN) continue;
        for (auto [v, w] : graph[u]) {
            if (dist[v] < dist[u] + w) {
                dist[v] = dist[u] + w;
            }
        }
    }

    return dist;
}
